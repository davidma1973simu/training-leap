# Training Leap

课后行动画布 · 基于福格行为模型的中高管培训后行动规划工具

## 功能

- **学员端** (`index.html`)：创建和管理课后行动计划
- **后台管理** (`admin.html`)：配置培训项目、目标受众、方法工具清单

## 本地预览

```bash
npx serve .
```

然后打开 http://localhost:3000/index.html
